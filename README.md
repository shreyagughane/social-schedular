# Social Scheduler
> AI-powered social media scheduling platform built with React, Node.js, Express, MongoDB, Gemini AI, and Cloudinary.
A full-stack social media scheduling platform that helps users create, manage, and schedule posts across multiple platforms.
> It also supports AI-powered post generation and media handling for faster content creation.

## Features
- Create and schedule social media posts
- AI-generated post captions using Gemini API
- AI image generation support
- Upload media for posts
- Connect and manage multiple social accounts
- View post history and AI generation history
- Responsive dashboard UI

## Tech Stack
### Frontend
- React.js
- TypeScript

- Tailwind CSS
### Backend
- Node.js
- Express.js
- TypeScript
